Image Classification API

Our API allows you to classify images and get predictions with confidence scores. Simply upload an image and receive instant analysis.

Endpoint

POST /check

Request

Send a POST request with form-data containing your image file.

PNG
JPG
JPEG
GIF
cURL
curl --location --request POST 'https://api.example.com/check' \ --form 'file=@"path/to/your/image.jpg"'

Response

Successful requests will return a 200 status code with prediction data:

{"message":"File analysed successfully","output":{"confidence":1.0,"prediction":"real"}}
prediction
"game" or "real"
confidence
confidence score (0-1)

Status Codes

200
Success - Image processed successfully
400
Bad Request - Invalid file or no file provided
415
Unsupported Media Type - File format not supported
500
Internal Server Error - Something went wrong on our end
